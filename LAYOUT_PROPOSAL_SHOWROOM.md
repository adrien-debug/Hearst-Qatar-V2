# Proposition de Layout : "Showroom & Production" (58 Conteneurs)

Ce layout est conçu pour allier **esthétique** (pour les visites) et **efficacité** (pour atteindre la cible de 58 conteneurs).

## 1. Concept Global

Le site est divisé en deux zones distinctes :
*   **La Zone "Vitrine" (Rangée 1)** : La première rangée visible en entrant. Elle est parfaitement symétrique, aérée et conçue pour accueillir les visiteurs ou servir de modèle de référence.
*   **La Zone "Production" (Rangées 2, 3, 4)** : Densifiée pour maximiser le nombre de machines et atteindre l'objectif de capacité.

## 2. Détail des Rangées

| Rangée | Type | Configuration (Transfos) | Conteneurs | Description |
| :--- | :--- | :--- | :--- | :--- |
| **Rangée 1** | **Showroom** | **3 Gauche / 3 Droite** | **12** | Symétrie parfaite. Espacement standard. Idéal pour les visites. |
| **Rangée 2** | Production | **4 Gauche / 4 Droite** | **16** | Densité maximale. Symétrie conservée. |
| **Rangée 3** | Production | **4 Gauche / 4 Droite** | **16** | Densité maximale. Symétrie conservée. |
| **Rangée 4** | Ajustement | **3 Gauche / 4 Droite** | **14** | Ajustement en fond de site pour atteindre le compte exact. |
| **TOTAL** | | **29 Transfos** | **58** | Objectif atteint. |

## 3. Schéma de Disposition

### Rangée 1 : "Showroom" (Vue de dessus)
Une disposition claire et équilibrée.
```
[Cont] [Cont] [Cont] --- [PB + Plateforme] --- [Cont] [Cont] [Cont]
(-30m) (-20m) (-10m)          (Centre)         (+10m) (+20m) (+30m)
```

### Rangées 2 & 3 : "Production Max"
On ajoute un bloc de chaque côté pour densifier.
```
[Cont] [Cont] [Cont] [Cont] --- [PB + Plateforme] --- [Cont] [Cont] [Cont] [Cont]
(-40m) (-30m) (-20m) (-10m)          (Centre)         (+10m) (+20m) (+30m) (+40m)
```

### Rangée 4 : "Fond de Site"
Léger décalage invisible depuis l'entrée principale.
```
       [Cont] [Cont] [Cont] --- [PB + Plateforme] --- [Cont] [Cont] [Cont] [Cont]
       (-30m) (-20m) (-10m)          (Centre)         (+10m) (+20m) (+30m) (+40m)
```

## 4. Éléments Communs
*   **Plateforme Béton Centrale** : Présente sur **toutes** les rangées (x=0) pour maintenir une continuité visuelle ("Spin" centrale).
*   **Symétrie Visuelle** : L'asymétrie de la dernière rangée est cachée par les rangées 2 et 3, préservant l'aspect "parfait" du site vu depuis l'entrée.
