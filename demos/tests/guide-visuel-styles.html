5. Typographie

Famille de Police

font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, sans-serif;

Tailles de Police

H1 - 1.75rem (28px)

Titre Principal

H2 - 1.25rem (20px)

Titre de Section

H3 - 1rem (16px)

Sous-section

Texte normal - 0.875rem (14px)

Texte normal avec couleur grise pour le contenu secondaire.

Petit texte - 0.75rem (12px)

Petit texte pour les labels, badges, métadonnées.

Poids de Police

Extrabold (800) - Pour les titres Hardware
Bold (700) - Pour les titres principaux
Semibold (600) - Pour les sous-titres
Medium (500) - Pour le texte important
Normal (400) - Pour le texte standard

Letter Spacing (Tracking)

tracking-wide (0.05em)
Titres principaux et sections
tracking-tight (-0.02em)
Titres Hardware et Container Detail
