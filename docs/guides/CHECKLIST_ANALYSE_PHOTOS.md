# Checklist d'Analyse des Photos

Utilisez cette checklist pour analyser systématiquement chaque photo fournie.

## Module Inférieur - Conteneur Pelin

### Structure générale
- [ ] Longueur exacte (mesurer ou estimer par rapport aux standards)
- [ ] Largeur exacte
- [ ] Hauteur exacte
- [ ] Forme générale (parallélépipède, angles droits, etc.)

### Matériau et texture
- [ ] Type de tôle (ondulée verticale, horizontale, lisse)
- [ ] Nombre de nervures/ondulations
- [ ] Espacement entre les nervures
- [ ] Couleur exacte (blanc, gris clair, autre ?)
- [ ] Texture de surface (mat, brillant, métallique)

### Coins et renforcements
- [ ] Présence de coins renforcés
- [ ] Dimensions des renforcements
- [ ] Couleur des renforcements
- [ ] Position exacte (aux 4 coins ?)

### Marquages et logos
- [ ] Logo ANTSPACE :
  - [ ] Position exacte (centre ? décalé ?)
  - [ ] Hauteur depuis le bas
  - [ ] Largeur
  - [ ] Forme exacte (A stylisé, antennes, etc.)
- [ ] Texte "BITMAIN" :
  - [ ] Position exacte (coin inférieur gauche ?)
  - [ ] Distance depuis le bord gauche
  - [ ] Distance depuis le bas
  - [ ] Taille du texte
- [ ] Texte "HD5" :
  - [ ] Position exacte (coin inférieur droit ?)
  - [ ] Distance depuis le bord droit
  - [ ] Distance depuis le bas
  - [ ] Taille du texte

### Éléments fonctionnels
- [ ] Protubérance côté gauche :
  - [ ] Position exacte (milieu-gauche ?)
  - [ ] Dimensions (longueur, largeur, hauteur)
  - [ ] Forme (rectangulaire, arrondie ?)
  - [ ] Grille d'aération :
    - [ ] Position sur la protubérance
    - [ ] Nombre de barres
    - [ ] Orientation (verticale, horizontale ?)
    - [ ] Espacement
- [ ] Grille d'aération coin inférieur gauche :
  - [ ] Position exacte
  - [ ] Dimensions
  - [ ] Orientation (verticale, horizontale ?)
  - [ ] Nombre de barres
- [ ] Verrous de conteneur :
  - [ ] Nombre de verrous
  - [ ] Position de chaque verrou
  - [ ] Dimensions
  - [ ] Forme

## Module Supérieur - Structure avec Radiateur en V

### Structure générale
- [ ] Longueur (identique au conteneur ?)
- [ ] Largeur (identique au conteneur ?)
- [ ] Hauteur (proportion par rapport au conteneur)
- [ ] Alignement avec le conteneur (parfaitement aligné ?)

### Cadre extérieur
- [ ] Couleur
- [ ] Texture (lisse, texturé ?)
- [ ] Épaisseur des parois
- [ ] Présence de montants/structures de support

### Panneaux en V (chevrons)
- [ ] Nombre exact de panneaux
- [ ] Dimensions de chaque panneau :
  - [ ] Largeur
  - [ ] Hauteur
  - [ ] Profondeur/épaisseur
- [ ] Angle d'inclinaison de chaque panneau
- [ ] Motif d'alternance (V inversé, chevrons)
- [ ] Couleur exacte (bleu-gris foncé ? code hex ?)
- [ ] Matériau (métallique, plastique ?)
- [ ] Espacement entre les panneaux
- [ ] Structures de support :
  - [ ] Nombre
  - [ ] Position
  - [ ] Dimensions
  - [ ] Couleur

### Extrémité gauche - Compartiment de ventilation
- [ ] Dimensions du compartiment
- [ ] Position exacte
- [ ] Ailettes de radiateur :
  - [ ] Nombre
  - [ ] Dimensions
  - [ ] Orientation (verticale ?)
  - [ ] Espacement
  - [ ] Couleur
- [ ] Tubes et tuyaux :
  - [ ] Nombre
  - [ ] Diamètre
  - [ ] Position de chaque tuyau
  - [ ] Direction (entrant, sortant ?)
  - [ ] Couleur (argenté, métallique ?)
  - [ ] Connexions vers les radiateurs en V

### Extrémité droite
- [ ] Poignée/élément d'accès :
  - [ ] Position exacte (coin supérieur droit ?)
  - [ ] Dimensions
  - [ ] Forme (rectangulaire ?)
  - [ ] Couleur (foncé ?)
  - [ ] Distance depuis les bords

### Tuyaux au-dessus des radiateurs
- [ ] Nombre de tuyaux
- [ ] Position de chaque tuyau
- [ ] Diamètre
- [ ] Longueur
- [ ] Direction (horizontale ?)
- [ ] Connexions (d'où viennent-ils ?)
- [ ] Couleur

### Ventilateur sur le dessus
- [ ] Position exacte (vers l'extrémité gauche ? centre ?)
- [ ] Dimensions (diamètre)
- [ ] Hauteur
- [ ] Structure (grille, ailettes ?)
- [ ] Couleur
- [ ] Détails visuels (structure en croix, grille, etc.)

## Relations entre les modules

- [ ] Alignement parfait (même longueur, même largeur ?)
- [ ] Position de superposition (centré ? décalé ?)
- [ ] Espace entre les modules (collés ? espace ?)
- [ ] Connexions visibles entre les modules
- [ ] Ligne de démarcation (nette ? floue ?)

## Éclairage et couleurs

- [ ] Couleur de fond (blanc pur ?)
- [ ] Éclairage (uniforme ? directionnel ?)
- [ ] Ombres visibles
- [ ] Reflets sur les surfaces métalliques

## Détails supplémentaires

- [ ] Autres éléments visibles non mentionnés
- [ ] Textures spéciales
- [ ] Marquages supplémentaires
- [ ] Éléments fonctionnels non listés
